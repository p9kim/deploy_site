Cloud Surf  ➩

An API that allows users to store files on the cloud. This was built using two different types of servers, three metadata servers and a block server. The metadata server would keep track of information of files such as the name, version, and blocklist. Depending on the operation metadata server would essentially communicate with block server to see if a file’s blocks of data are stored on the block server, but before all this version numbers are checked with the metadata server so race conditions do not occur.

gRPC Java

Chat Analysis Bot  ➩

A sentiment analysis bot that scans Twitch chat for patterns to deliver better service to the users developed at UCSD Hackathon. Using queue data structure of fixed size, the queue will be filled with texts from users in chat. That queue will be analyzed to see if there is high percentage of the same “reaction” and if threshold is passed then additional content or service will be provided. A direct application of this would be instant replays like in sports when a crazy play or a player does something amazing.

Python Twitch API

Send It  ➩

This project is to help me learn the programming language Rust along with Rust WebAssembly. Using TCP Sockets create a website that connects two users peer to peer to allow them to send files to each other directly and fluently. OR due to potential security issues for users, instead make it so users upload a file that has an expiration time specified by the uploader (max 24 hours). This generates a link that can be shared with others allowing them to download the file until expiration time. When expiration time is up link and file will be deleted from the server.
